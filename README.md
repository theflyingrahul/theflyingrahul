### 📌 Featured Projects

Flagship open-source work in **search, information retrieval, and AI evaluation**.

**🔎 [LISA — Literature Intelligence & Search Architecture](https://github.com/theflyingrahul/lisa)**
Turns a folder of scholarly PDFs into a structured, cross-linked, queryable knowledge base — with evidence traceable down to the supporting sentence. A hybrid engine fuses lexical (BM25), dense, structured, and graph search with cross-encoder reranking, plus a knowledge graph, UMAP corpus map, and CLI / API / web / MCP interfaces. Hardware-accelerated with OpenVINO (3.3× embedding, 11× reranking).
`Python 3.11+` · `SQLite/FTS5` · `DuckDB` · `fastembed` · `cross-encoders` · `OpenVINO/CUDA` · `MCP`

**🌀 [dow — Drift Observation Workbench](https://github.com/theflyingrahul/dow)**
"Git for AI behavior" — version your complete inference spec (prompt, model identity, sampling, eval config), then measure semantic drift, stability, and regressions between versions, with causal attribution down to the exact config field responsible. Ships a fail-closed CI regression gate and an MCP server exposing 16 tools; runs fully offline by default.
`Python 3.9+` · `Git` · `MCP` · `sentence-transformers` · `OpenAI/Ollama/vLLM`

**🕸️ [dgit — Peer-to-Peer Distributed Version Control](https://github.com/theflyingrahul/dgit)**
A decentralized VCS that distributes repository history across peers instead of a central host. Design and initial insights presented at **DESRIST 2023**.
`Python` · `P2P`
